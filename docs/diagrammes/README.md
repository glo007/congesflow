# Diagrammes UML — CongésFlow

Ce dossier contient les sources des diagrammes du dossier projet, dans deux formats :

- `.mmd` → **Mermaid** (classes, séquence, état, activité, MCD)
- `.puml` → **PlantUML** (contexte, cas d'utilisation, objets, composants, déploiement)

## Comment générer les images (PNG/SVG) pour le mémoire

**Mermaid (.mmd)**
- En ligne : copier-coller dans <https://mermaid.live> → exporter en PNG/SVG.
- VS Code : extension *Markdown Preview Mermaid Support*.

**PlantUML (.puml)**
- En ligne : <https://www.plantuml.com/plantuml> → coller le code.
- VS Code : extension *PlantUML* (aperçu avec `Alt+D`).
